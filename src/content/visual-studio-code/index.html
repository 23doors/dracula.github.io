Visual Studio Code

Install using Command Palette

  1. Go to View -> Command Palette or press Ctrl+Shift+P
  2. Then Enter Install Extension
  3. Write theme-dracula
  4. Select it or press Enter to install

Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo:

$ git clone https://github.com/zenorocha/dracula-theme/

Install manually

Download using the GitHub .zip download option and unzip them.

Activating theme

Copy the theme-dracula folder to your .vscode/extensions folder and run Visual Studio Code. The Dracula theme will be available from File -> Preferences -> Color Theme dropdown menu.
