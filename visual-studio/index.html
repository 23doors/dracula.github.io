Visual Studio

Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo:

$ git clone https://github.com/zenorocha/dracula-theme/

Install manually

Download using the GitHub .zip download option and unzip them.

Activating theme

Start Visual Studio and go to Tools -> Import and Export Settings... -> Import selected environment settings -> Choose to backup existing settings or not -> Click browse and select dracula.vssettings -> Finish
